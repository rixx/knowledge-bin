# Lists

## General structure

    \begin{list_type}

    \item bla
    \item keks

    \end{list_type}

## List Types

* Itemize: bullet points
* Enumerate: yup, that.
* Description: `\item[First] \hfill \\ The first item`

## Nesting:

    \begin{enumerate}
      \item first
      \begin{enumerate}
        \item alpha
      \end{enumerate}
    \end{enumerate}
